## 💡 Instrucciones de Uso (One-liners)

*   **Stitch UI:** Maquetación rápida y componentes visuales Google-like.
*   **Browser Control:** Navegación dirigida, clics y scrapping selectivo.
*   **Smart Reviewer:** Feedback sobre lógica, deuda técnica y seguridad.
*   **TypeScript:** Resolución de tipos complejos y perf de compilación.
*   **API Security:** JWT, sanitización y protección contra inyecciones.
*   **Docker:** Imágenes multi-stage ligeras y hardening.
*   **Clean Code:** Transforma código funcional en código mantenible.
*   **n8n Ecosystem:** Suite completa para automatización (Tools, Config, Validation).
*   **n8n Code:** JavaScript (recomendado) y Python para transformaciones complejas.
*   **n8n Expressions:** Paso de datos impecable entre nodos (evita errores `.body`).
*   **Food Query:** Macros, micros y recomendaciones nutricionales basadas en datos.
*   **RecallMax:** Memoria infinita (hasta 1M tokens) sin alucinaciones.
*   **Design Expert:** Dashboards de alta gama con GSAP, glassmorphism y profundidad 3D.
*   **Viral Growth:** Estrategias de referidos y cálculo de LTV/CAC para crecimiento.
*   **Brainstorming:** Revisión extrema estructurada con Escéptico, Guardián y Árbitro.
*   **Elite Web Design Suite:** Conjunto de 10 skills para alcanzar el nivel de Awwwards/FWA:
    *   **GSAP:** El estándar de oro para animaciones premium (ScrollTrigger, Flip, SplitText).
    *   **Accessibility:** Animaciones inclusivas que respetan `prefers-reduced-motion`.
    *   **Performance:** Velocidad extrema para que las animaciones nunca pierdan fluidez (60fps).
    *   **Layouts:** Maquetación moderna y editorial (Bento, Parallax, Sticky, Horizontal).
    *   **Brand Design:** Traduce la personalidad de marca en decisiones de diseño coherentes.
    *   **Design Core:** Cimientos para que un sitio parezca diseñado por un estudio de primer nivel.
    *   **CSS Animations:** Animaciones nativas ligeras (Scroll-Driven, View Transitions).
    *   **UX Strategy:** Asegura que el diseño no solo sea bonito, sino que también convierta.
    *   **Inspiration:** Biblioteca de patrones y casos de éxito de sitios premiados.
    *   **Audit:** El guardián final de la calidad UI/UX (~190 reglas de oro).

## ⚡ Modo de Ahorro (Optimización de Tokens)

Usa estos parámetros para evitar escaneos completos del repositorio:

1.  **Smart Reviewer / A11y / Clean:** Especifica siempre la ruta (ej: `/clean src/index.js`).
2.  **TypeScript:** Usa `--fast-check` para diagnósticos rápidos.
3.  **Secure API:** Pide `--checklist` para una revisión puntual.
4.  **n8n Suite:** Usa `--no-nodes` para arquitectura o `--validate-only` para errores.
5.  **n8n Code:** Pide `--patch-only` si solo quieres cambiar una línea específica.
6.  **RecallMax:** Usa `--summary-only` para compresión de hilo.
7.  **Design Expert:** Usa `--motion-only` para enfocarte solo en animaciones GSAP.
8.  **Nutrition:** Pide `--macros-only` si no necesitas desglose de vitaminas.
9.  **Brainstorming:** Usa `--role-skeptic` para forzar solo modo de crítica destructiva.

# Smart Reviewer Skill

Realiza revisiones de código profundas, auditorías de seguridad y análisis de deuda técnica.

## Instrucciones
- Analizar consistencia y calidad del código.
- Identificar posibles bugs y vulnerabilidades.
- Sugerir mejoras de rendimiento y mantenibilidad.

## Modo de Ahorro
- Indica archivos o carpetas específicas (e.g., `/review src/utils.js`) para evitar que el skill analice todo el repositorio.

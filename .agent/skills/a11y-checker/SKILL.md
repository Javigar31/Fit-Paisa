# A11y Checker Skill

Especializado en asegurar que las aplicaciones web cumplan con los estándares internacionales de accesibilidad.

## Instrucciones
- Ejecutar tests de accesibilidad basados en WCAG.
- Detectar problemas de contraste y falta de etiquetas ARIA.
- Sugerir o aplicar correcciones automáticas.

## Modo de Ahorro
- Ejecuta con el flag `--audit-only` para recibir el reporte de errores sin que el agente intente aplicar correcciones automáticas de inmediato.

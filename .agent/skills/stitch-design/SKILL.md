# Stitch Design Skill

Este skill permite integrar el flujo de trabajo de Google Stitch UI para generar prototipos y componentes front-end rápidos.

## Instrucciones
- Usa componentes del sistema de diseño de Google.
- Genera wireframes basados en prompts de usuario.
- Convierte diseños de Stitch en código React o HTML/CSS.

## Modo de Ahorro
- Llama con `--no-analyze` para evitar el escaneo completo del repositorio antes de comenzar la tarea.

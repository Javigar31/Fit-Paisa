# Browser Control Skill

Permite al agente interactuar con el navegador web de forma directa para tareas de automatización y búsqueda.

## Instrucciones
- Navegar a URLs específicas.
- Interactuar con elementos del DOM (hacer clic, escribir texto).
- Extraer contenido y capturar pantallas.

## Modo de Ahorro
- Usa el parámetro `--shallow` para limitar la profundidad de análisis de las páginas y ahorrar tokens.

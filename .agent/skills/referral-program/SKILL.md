# Referral & Viral Growth Expert

## Program Strategy
- **Customer Referral**: Existing users recommending to friends (Trust-based).
- **Affiliate Program**: Creators/Partners earning commissions (Volume-based).
- **Hybrid**: Both for maximum reach.

## The Referral Loop
1. **Trigger Moment**: Post-"Aha" moment, milestone, or renewal.
2. **Share action**: In-product, personalized links, codes.
3. **Convert Referred**: Optimize the landing experience.
4. **Reward**: Double-sided (both get $), Tiered (milestones), fixed (store credit).

## Incentive Framework
`Max Reward = (Customer LTV * Gross Margin) - Target CAC`

- **SaaS B2B**: $50-500 or 1-3 months free.
- **Consumer B2C**: $10-50 or 15-25% discount.

## Metrics to Track
- **K-factor (Viral Coeff)**: `Invitations * Conversion Rate`. K > 1 = Viral.
- **Referral Rate**: Targets: 10% (Good), 25% (Great), 50% (Exceptional).
- **ROI**: `(Revenue from Referred - Program Costs) / Program Costs`.

## Fraud Prevention
- Email verification + Device fingerprinting.
- IP monitoring + Minimum activity thresholds.
- Delayed rewards (pay out after 30 days of active use).
